# **App Name**: Raseed AI

## Core Features:

- Intelligent Receipt Scanner: AI-powered receipt data extraction from uploaded images or videos using optical character recognition.
- AI Financial Assistant: Users can query their spending habits via a conversational UI; a tool analyzes the query and user spending data to answer the user.
- Smart Investment Recommendations: Provide personalized smart investment recommendations based on user inputs and risk profile.
- Home Dashboard: Interactive dashboard displaying monthly spending, savings, and investment suggestions.
- Budget & Expense Tracker: Expense tracker with categorized spending and location-based comparison.
- Receipt Vault: Vault to list and organize all receipts with search and filter functionality.
- My Wallet Passes: Wallet Passes: Generate shareable summaries for receipts, budget tips, and investment plans.

## Style Guidelines:

- Primary color: Google blue (#4285F4) to reflect the brand's identity and instill trust. This will be the main color for UI elements and interactions.
- Background color: Light gray (#F5F5F5) to provide a clean and modern feel. A very subtle tint of blue has been added.
- Accent color: Soft green (#34A853) used to highlight positive actions like savings and investment gains.
- Body and headline font: 'PT Sans' (sans-serif) combines a modern look and a little warmth.
- Consistent usage of Google Material Design icons to provide a familiar and intuitive experience.
- Adhere to Google Material Design responsive layout principles for a seamless user experience on various devices.
- Subtle and purposeful animations to enhance user interactions and provide feedback, such as loading animations or transition effects.
